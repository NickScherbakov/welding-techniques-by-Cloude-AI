# Как внести вклад

Спасибо за интерес к проекту! Мы приветствуем вклад как от ученых, так и от специалистов‑практиков.

## Формы вклада
- Статья/отчет об исследовании (раздел «Статьи и опыт»)
- Практическое руководство, чек‑лист, параметры режимов
- Поправки и дополнения к существующим страницам

## Быстрый процесс
1. Форкните репозиторий и создайте ветку из `main`.
2. Для статьи используйте шаблон из «Шаблоны материалов».
3. Локально соберите сайт, убедитесь, что все ссылки и формулы отображаются.
4. Откройте Pull Request: опишите суть, дайте краткую аннотацию и ключевые слова.

## Локальная сборка
```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
mkdocs serve
```
Сайт откроется на http://127.0.0.1:8000

## Стиль и оформление
- Ясная структура: введение → методика → результаты → обсуждение → выводы
- Иллюстрации: подписи, единицы измерения, читаемость в темной/светлой теме
- Формулы: используйте KaTeX (`$...$` и `$$...$$`)
- Ссылки на источники: по возможности с DOI или ссылками на открытый доступ

## Review
Каждый PR проходит техническую проверку (сборка/навигация) и содержательную (корректность, воспроизводимость). Просим отвечать на вопросы ревьюверов и при необходимости вносить правки.

См. также: «Кодекс поведения».
