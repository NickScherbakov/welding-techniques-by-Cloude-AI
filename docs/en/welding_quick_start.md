# Quick Start

This repository aggregates welding techniques documentation. Translation of extended Russian quick start guide will follow. For now:

## Main Sections

- **Theory**: Arc physics, metallurgy fundamentals.
- **Practice**: Equipment, parameters, setup workflow.
- **Troubleshooting**: Common defects & corrective actions.
- **Articles & Experience**: Case studies, experiments, novel approaches.

## Contributing

1. Fork the repository.

3. Add or translate content under appropriate folder.
4. Open a Pull Request describing changes.
