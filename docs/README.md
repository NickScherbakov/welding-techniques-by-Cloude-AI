
# Сообщество сварщиков — Cloude AI

Сайт для энтузиастов, ученых и специалистов-практиков сварки. Здесь делятся опытом, теорией, методиками, параметрами режимов и реальными кейсами.

## О проекте
Этот репозиторий теперь — основа для сайта сообщества сварщиков. Все материалы доступны онлайн, можно легко дополнять и обсуждать.

## Как запустить сайт локально
```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
mkdocs serve
```
Откроется на http://127.0.0.1:8000

## Как внести вклад
1. Ознакомьтесь с [правилами и шаблонами](submit_guidelines.md).
2. Форкните репозиторий, создайте ветку.
3. Добавьте статью или правки.
4. Откройте Pull Request, заполните шаблон.

## Структура сайта
- Главная: index.md
- Быстрый старт: welding_quick_start.md
- Теория: welding_theory.md
- Практика: оборудование, параметры, наладка
- Материалы: файлы, изображения
- Трудности: типовые проблемы
- Статьи и опыт: оригинальные исследования, кейсы
- Сообщество: вклад, шаблоны, кодекс поведения
