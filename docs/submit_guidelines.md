# Шаблоны материалов и правила публикации

Чтобы упростить ревью и сделать материалы сопоставимыми, используйте следующий шаблон.

## Шаблон статьи (исследование/методика)
- Название
- Краткая аннотация (2–4 предложения)
- Ключевые слова
- Введение (контекст, постановка задачи)
- Методика (оборудование, материалы, параметры, дизайн эксперимента)
- Результаты (графики/таблицы/фото, основные наблюдения)
- Обсуждение (интерпретация, ограничения, сравнение с литературой)
- Выводы (что применимо на практике)
- Ссылки (DOI/ссылки на открытый доступ)
- Приложение (сырые данные, скрипты, дополнительные фото) — опционально

## Технические требования
- Формулы: KaTeX (`$...$`, `$$...$$`)
- Изображения: `images/` или ссылки; указывайте источники
- Единицы: SI; четкие подписи
- Повторяемость: параметры режимов, настройки оборудования, детали подготовки образцов

## Как отправить
1. Создайте ветку и добавьте файл в `articles/` (пример: `articles/2025-11-исследование-GMAW-импульс.md`).
2. Заполните разделы по шаблону.
3. Откройте Pull Request и приложите превью (скриншоты/рендер локального сайта).
4. При необходимости создайте Issue «Предложить статью» для обсуждения перед PR.
